# Push Axon to GitHub

Create an empty GitHub repository named `axon-student-portal` without initializing it with a README, license, or `.gitignore`.

Then run from this project directory:

```bash
git remote add origin https://github.com/YOUR_USERNAME/axon-student-portal.git
git branch -M main
git push -u origin main
```

If an `origin` remote already exists, replace it:

```bash
git remote set-url origin https://github.com/YOUR_USERNAME/axon-student-portal.git
git push -u origin main
```
